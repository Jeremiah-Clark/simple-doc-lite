# Appendix

Images are centered automatically and default to 80% of the text width.
Give them a width if you want something else, and the alt text becomes a
caption under the image:

```markdown
![Quarterly results chart](../images/chart.png){ width=60% }
```

For a manual page break anywhere, put `\newpage` on its own line.

That's everything you need to start — delete these starter files when
you're ready, and see the README for the full configuration reference.
